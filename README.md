# Fundamentals Project

## Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)

## Setup & Running

### 1. Install dependencies
```bash
npm install
```

### 2. Compile SASS and launch the project
```bash
npm run dev
```

This command compiles all `.scss` files from `src/scss` into `.css` files 
placed in the `dist` folder, then launches a local development server.

## Scripts
- `npm run compile` – compiles `.scss` files into `.css`
- `npm run dev` – compiles SASS and starts the development server
